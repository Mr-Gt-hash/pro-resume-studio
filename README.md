# ProResume Studio 📄⚡

> A sleek, modern, interactive web-based resume builder with live dual-pane rendering, custom themes, real-time ATS scoring, and 1-click clean PDF export.

## 🌟 Key Features

- **⚡ Real-Time Dual-Pane Editor**: Instant live preview as you type your personal details, summary, work history, skills, projects, and certifications.
- **🎨 Custom Templates & Themes**:
  - **Executive Modern**: Clean grid structure with subtle accent borders.
  - **Tech Minimalist**: Centered header layout optimized for engineers & developers.
  - **Creative Modern**: Distinct sidebar split layout with badge tags.
- **🎯 Live ATS Optimization Engine**: Instant feedback scoring action verbs, key technical terms (AWS, Docker, Kubernetes, OWASP, Penetration Testing), and structural balance.
- **📄 1-Click Clean PDF & HTML Downloads**: Direct client-side PDF generation via `html2pdf.js` with zero watermarks and `page-break-inside: avoid` rules.
- **💾 Data Backup & GitHub Integration**: Export/import resume state in JSON format or pull public repositories directly from GitHub.

## 🚀 Getting Started

No heavy build steps or node dependencies required! ProResume Studio runs natively in any browser.

```bash
# Clone the repository
git clone https://github.com/Mr-Gt-hash/pro-resume-studio.git

# Navigate to project directory
cd pro-resume-studio

# Open in browser or serve locally
python -m http.server 3030
```

Open `http://localhost:3030` in your web browser.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom responsive glassmorphism UI, flexbox/grid layouts, and `@media print` rules.
- **Vanilla JavaScript (ES6+)**: Reactive state management, dynamic DOM rendering, and ATS evaluation logic.
- **html2pdf.js**: High-definition HTML-to-PDF rendering.
